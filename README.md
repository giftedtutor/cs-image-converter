# CS Image Converter

Free browser-based image converter, resizer, and editor by Code Splitters.

## Features

- Convert between PNG, JPEG, WebP, GIF, BMP, AVIF, and ICO
- Resize with custom dimensions or social/media presets
- Rotate, flip, brightness, contrast, saturation, hue, blur
- Filters: grayscale, sepia, invert
- Side-by-side original vs converted preview
- Batch convert multiple images
- 100% client-side — images never leave your device

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

Optional: set `NEXT_PUBLIC_SITE_URL` for canonical URLs, Open Graph, sitemap, and robots (defaults to `https://csimageconverter.com`).
